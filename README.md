# jordanmajd.com

A metalsmith site to share my thoughts and projects.

## About

To install:

```bash
npm install
```

To build and run development server:

```bash
npm start
```

To build for production:

```bash
npm run build
```

To configure global variables, modify [src/config.json](/src/config.json).
